# PocketWise – Simple Finance App

PocketWise is a beginner-friendly personal finance app designed for young professionals. It helps users easily track income and expenses, monitor savings goals, and understand spending behavior through simple insights.

## Features
- Add income and expenses
- Prevent overspending (no negative balance)
- Track savings goals with progress
- Money Health insights (spending analysis & suggestions)
- Simple and clean user interface

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
Open pocket.html in your browser
